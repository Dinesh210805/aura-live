✅ How It Handles Amazon Cart:

  1. After typing: Observes search results screen
  2. LLM Reasoning: "See search suggestions + results. Need to tap actual product listing."
  3. Smart Selection: Uses VLM to distinguish suggestion vs result
  4. After tapping: Observes again - "Am I on product page?"
  5. Verification: Checks for product details, price, Add to Cart button
  6. If wrong screen: Goes back, tries different result
